# AI Update Prompt

Use this prompt when asking an AI to update SitePilot AI.

```text
You are updating the open-source SitePilot AI repo.

Preserve the existing PWA structure.

Do not remove:
- index.html
- manifest.webmanifest
- sw.js
- icons/icon.svg
- GitHub Pages workflow
- README.md
- CONTRIBUTING.md

Improve the requested area while keeping the app phone-friendly and usable offline.

Construction safety rules:
- Do not invent standards, tolerances, inspection requirements, or test frequencies.
- Mark unconfirmed requirements as needing formal confirmation.
- Do not imply AI replaces formal approval, engineering, certification, contract requirements, or client/superintendent direction.

After editing, provide:
1. Summary of changes
2. Files changed
3. Testing steps
4. Any risks or follow-up tasks
```
