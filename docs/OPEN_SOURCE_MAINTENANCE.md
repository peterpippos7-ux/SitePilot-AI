# Open Source Maintenance

## Branches

- `main` — stable published version
- `update/construction-pack-vX` — Construction Pack updates
- `feature/<name>` — new features
- `fix/<name>` — bug fixes

## Release Process

1. Update app files.
2. Update `CHANGELOG.md`.
3. Test locally with `python -m http.server 8080`.
4. Commit.
5. Open PR.
6. Merge to `main`.
7. GitHub Pages deploys automatically.
