# Contributing to SitePilot AI

Thanks for helping improve SitePilot AI.

Useful contributions include:

- better RFI workflows
- improved ITP and checklist templates
- defect closeout improvements
- site diary improvements
- programme risk workflows
- mobile UI improvements
- export and backup improvements
- safer AI prompt rules
- documentation improvements

## Core Rules

Do not add features that:

- invent standards or compliance requirements
- imply AI replaces formal approvals
- hide assumptions
- silently send project data to cloud AI
- commit API keys or secrets

## Pull Request Checklist

- [ ] The app still opens on mobile.
- [ ] Records still save locally.
- [ ] Exports still work.
- [ ] Text remains readable on a phone.
- [ ] No secrets are committed.
- [ ] Unconfirmed construction requirements are marked as needing confirmation.
