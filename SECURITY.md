# Security Policy

SitePilot AI currently runs as a static PWA and stores records in the user's browser storage.

The current mobile version does not directly send records to an AI API. Users can copy generated prompts into their chosen AI tool.

Future versions that add direct AI API access should include:

- explicit user consent
- clear AI provider selection
- local mode
- data redaction controls
- no hard-coded API keys
- no silent cloud transmission of project data

## Construction Safety

Do not use this tool as a substitute for formal professional advice, engineering certification, statutory compliance, project specifications, or client/superintendent directions.
